# Twitter-Inspired Social Media Application

A social networking application inspired by Twitter where users can share posts, upload images, and interact with other users.

## Features

- User authentication
- Create and delete posts
- Like and comment system
- Follow and unfollow users
- Profile and cover image uploads
- Notifications feature
- Image uploads with Cloudinary
- Responsive UI

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- React Query
- Cloudinary
- Tailwind CSS
- JWT

## Installation

```bash
git clone https://github.com/diwakar-dev1/twitter-inspired-social-app.git
cd twitter-inspired-app
npm install
npm run dev
```
## Production Build

```bash
npm run build
npm start
```
